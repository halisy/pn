Data & settings

Where your people are stored.
Everything you add is auto-saved in this browser so the map just works day to day. But the durable copy — the one that is backed up and moves between devices — is the SEED_DATA block inside this file on GitHub. When you have added or changed people, click Copy data block and paste it over the old SEED_DATA in index.html, then commit. (Or just download the JSON as a backup.)

Change your passcode

Type a new passcode, copy the generated hash, and paste it as PASS_HASH at the top of index.html, then commit. Be honest with yourself about this: a passcode in a public web page only stops casual snooping — anyone who opens "view source" can read past it. For a truly private map, keep this file in a private repo and open it locally, or ask me to set up a host with a real login.
